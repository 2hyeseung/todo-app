# Todo App

## StartProject

```
git clone [url]
cd server/
npm i
node app.js
cd ../client
npm i
npm start
```

## Stack

-   Node.js, express
-   MySQL
-   React

## Directory Structure

```
client/ # react app
server/ # express app
.gitignore
```

## REST API

-   `GET /todos` - show all todos
-   `POST /todo` - create a new todo
-   `PATCH /todo/:todoId` - edit a specific todo
-   `DELETE /todo/:todoId` - delete a specific todo
